Automatizando o trabalho contábil com Python: por onde começar

28 de junho de 2026 · Victor Ramos
python contabilidade automação

Quem trabalha em escritório de contabilidade conhece a rotina: baixar extratos, conferir lançamentos linha a linha, digitar dados de um sistema para outro, montar a mesma planilha todo mês. Boa parte desse trabalho não exige julgamento profissional — exige paciência. E tudo que exige apenas paciência pode ser automatizado.

Neste post conto como comecei a usar Python no meu dia a dia contábil, sem formação em TI, e o que eu recomendaria para quem quer seguir o mesmo caminho.

Por que Python (e não só Excel)?

O Excel resolve muita coisa — e continuo usando todos os dias. Mas ele tem limites práticos: arquivos grandes ficam lentos, processos manuais dependem de quem executa, e repetir a mesma sequência de passos todo mês é convite para erro. Um script Python faz a mesma tarefa em segundos, do mesmo jeito, todas as vezes.

Para o trabalho contábil, três bibliotecas resolvem a maioria dos problemas:

  • pandas — ler, filtrar, cruzar e agregar dados tabulares (extratos, balancetes, relatórios de sistema);
  • openpyxl — ler e gerar arquivos Excel formatados, sem abrir o Excel;
  • pathlib / os — organizar arquivos em lote (renomear, mover, separar por competência ou por empresa).

Um exemplo real: conciliação de extratos

Um dos primeiros processos que automatizei foi a preparação de extratos bancários para conciliação. O fluxo manual era: baixar o extrato em CSV, abrir no Excel, padronizar datas e valores, classificar lançamentos recorrentes e montar a planilha no layout que o sistema contábil importa.

Com pandas, o núcleo desse processo cabe em poucas linhas:

import pandas as pd

# Lê o extrato exportado pelo banco
extrato = pd.read_csv("extrato_junho.csv", sep=";", decimal=",")

# Padroniza datas e valores
extrato["data"] = pd.to_datetime(extrato["data"], dayfirst=True)
extrato["valor"] = extrato["valor"].astype(float)

# Classifica lançamentos recorrentes por palavra-chave
regras = {"TARIFA": "Despesas bancárias", "PIX RECEB": "Recebimento de clientes"}
def classificar(hist):
    for chave, conta in regras.items():
        if chave in hist.upper():
            return conta
    return "A CLASSIFICAR"

extrato["conta"] = extrato["historico"].apply(classificar)

# Gera a planilha no layout de importação
extrato.to_excel("importacao_junho.xlsx", index=False)

O que levava perto de uma hora por empresa passou a levar segundos — e o tempo ganho vai para o que realmente exige um contador: analisar as exceções, revisar o que o script marcou como "A CLASSIFICAR" e fechar a conciliação.

Por onde começar

  • Escolha um processo repetitivo e pequeno — não tente automatizar o fechamento inteiro de uma vez;
  • Aprenda o básico de pandas com dados que você conhece (seu próprio extrato é um ótimo dataset);
  • O livro Automate the Boring Stuff with Python, do Al Sweigart, é o melhor ponto de partida — está na minha Estante;
  • Versione seus scripts no Git desde o início, mesmo trabalhando sozinho.
